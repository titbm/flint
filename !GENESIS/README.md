# !GENESIS

Папка содержит оригинальные файлы программы Flint для управления лазерным станком.

```
!GENESIS/
├── Flint 2.61/             — оригинальные файлы программы Flint 2.61
│   └── SRC/                — оригинальные исходные файлы (.CPP, .H)
│
└── Flint 2.77/             — оригинальные файлы программы Flint 2.77
```
---
<br>

## Требования

Рядом с папкой `!GENESIS/` должна находиться папка `Tools/` со следующими инструментами:

```
├── !GENESIS/
│
└── Tools/
    ├── DOSBox-X/      — эмулятор DOS (для запуска и компиляции на современном Windows)
    └── BC45/          — компилятор Borland C++ 4.5 (для создания exe из исходных файлов)
```
Для запуска программ достаточно только `DOSBox-X/`.

---
<br>


## Как запустить

Запустить `RUN.bat`: 

- `Flint 2.61\RUN.bat`
- `Flint 2.77\RUN.bat`

Скрипт автоматически определит среду и выполнит:

- **DOS / Windows 98** — запуск `PROJ5.EXE` напрямую
- **Современный Windows** — запуск `PROJ5.EXE` через DOSBox-X

На **DOS / Windows 98** работает запуск `PROJ5.EXE` напрямую

---
<br>

## Как создать .exe-файл Flint 2.61 из исходных файлов

Запустить `Flint 2.61/SRC/COMPILE.bat`. 

Скрипт автоматически определит среду и выполнит:

- **DOS / Windows 98** — компиляция нативно через Borland C++
- **Современный Windows** — компиляция через DOSBox-X с Borland C++ внутри

После завершения появится папка с программой `Flint 2.61/SRC/BUILD`

<br>

*Примечание:* cкрипт ищет Borland C++ в следующем порядке:

1. Переменная `BCDIR` (если задана заранее: `SET BCDIR=C:\BC45`)
2. `..\..\..\Tools\BC45` (относительно SRC)
3. `C:\BC45`

<br>


### Имена файлов на чистом MS-DOS

На чистом MS-DOS (без поддержки длинных имён файлов) папки вроде `Flint 2.61`, `!GENESIS` и т.д. не будут работать. Перед использованием на такой системе переименуйте папки в формат 8.3, например:

| Исходное имя         | Имя для DOS  |
|----------------------|-------------|
| `!GENESIS`           | `GENESIS`   |
| `Flint 2.61`        | `FLINT261`  |
| `Flint 2.77`        | `FLINT277`  |
| `Flint 2.77 src`    | `FL277SRC`  |

На Windows 98 длинные имена поддерживаются, переименование не нужно.
