# Содержимое папки Help

Папка содержит документацию к проекту FLINT: описание программы и оборудования, справочники по функциям и форматам файлов, скриншоты интерфейса, сканы принципиальных схем и справочную документацию по инструментам разработки.

Справочная документация по инструментам, использованным при разработке и портировании.

#### docs/Borland PowerPack for DOS 1.0a/

Документация по **Borland PowerPack for DOS 1.0a** — пакету расширений для Borland C++ 4.5, который используется в 32-битном порте `Flint 2.77 (BCC32) src/`. Пакет предоставляет компилятор `BCC32.EXE`, DPMI32-расширитель (`32RTM.EXE` + `DPMI32VM.OVL`) и графическую библиотеку BGI для 32-битных DOS-приложений (`BGI32.DLL`, `BGIVGA.DLL`).

Содержимое папки создано путём **обратного инжиниринга** оригинальных `.HLP`-файлов пакета (формат Windows Help) — они были извлечены и преобразованы в Markdown:

| Файл / папка | Содержание |
|--------------|------------|
| [ABOUT_POWERPACK_FOR_DOS.md](docs/Borland%20PowerPack%20for%20DOS%201.0a/ABOUT_POWERPACK_FOR_DOS.md) | README из дистрибутива: установка, состав пакета, список файлов |
| [BORLAND_POWERPACK_FOR_DOS_HELP_EXTRACTED.md](docs/Borland%20PowerPack%20for%20DOS%201.0a/BORLAND_POWERPACK_FOR_DOS_HELP_EXTRACTED.md) | Текст справочной системы PowerPack, извлечённый из `.HLP`-файлов |
| `examples/` | Примеры кода из дистрибутива (`DPMI/`, `DPMI16/`, `DPMI32/`, `TVISION/`) |
| `images/` | Изображения из оригинальной справки |
